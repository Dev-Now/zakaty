# Zakaty - Windows Desktop Application

## Installation Instructions
Launch the installation wizard and follow the instructions.

## System Requirements

- Windows 10 or later
- At least 100MB of free disk space
- 4GB RAM recommended

## Configuration

The application will create its configuration files automatically on first run.

If you want to customize supported currencies or set you own working directory you can edit the values in:
"%APPDATA%\com.example\zakaty\config.json"


## Usage

1. Launch the application from startup menu
2. The application will start and you can begin using it immediately

## Troubleshooting

If the application doesn't start:
- Make sure you have administrative rights on your computer
- Check if Windows Defender or your antivirus is blocking the application
